
---

description: Prepara e faz deploy do projeto no Vercel

---



Siga os passos para deploy:

1. Verifique se o requirements.txt está atualizado

2. Confirme que o vercel.json está correto

3. Execute: git add . && git commit -m "deploy: atualização"

4. Execute: git push origin main



Informe que o Vercel fará o deploy automaticamente após o push.
